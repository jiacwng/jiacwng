### Featured work

- **[Mnemocrypt Enhanced](https://github.com/jiacwng/mnemocrypt-enhanced)** — Extended an open-source ML-based IDA Pro plugin for detecting cryptographic functions in compiled binaries. Expanded its evaluation from x86 to x86-64, added SIMD- and operand-level features, and improved held-out goodware precision from 60% to 86% while increasing recall from 25% to 46%.
- **[Malfamily](https://github.com/jiacwng/malfamily)** — Built a static-analysis pipeline that uses Ghidra and Python to classify x86 PE and ELF samples into six family-derived malware categories. The current model reaches 77.9% accuracy and 0.77 macro-F1 on 472 deduplicated samples, with confidence and analyzability checks for uncertain inputs.
- **[CTF Write-ups](https://github.com/jiacwng/ctf-writeups)** — Maintain 25+ technical write-ups covering reverse engineering, binary exploitation, digital forensics, cryptography, and web security across picoCTF, Root-Me, and TryHackMe.
